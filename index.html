This is T-Pose Soccer

Features

T-Pose Soccer is a classic 2-player game for PC where fun is the core of gameplay.

  • Play with your friend with the same keyboard. What could be more fun?
  • Anything can happen when there are randomly generated mystery boxes around the soccer field.
  • You can choose how many rounds you want to play.
  • Endless fun!

Download T-Pose Soccer

T-Pose Soccer is on alpha stage currently only for Windows and is available for free to download. New features are added constantly. So go ahead and download. What are you waiting for?

Tools

T-Pose Soccer is made with Unity game development platform, C# with Visual Studio 2017 and MagicaVoxel voxel art editor.
