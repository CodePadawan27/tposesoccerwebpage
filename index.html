Features

T-Pose Soccer is a classic 2-player game for PC where fun is the core of gameplay.

  • Play with your friend with the same keyboard. What could be more fun?
  • Anything can happen when there are randomly generated mystery boxes around the soccer field.
  • You can choose how many rounds you want to play.
  • Endless fun!

Download T-Pose Soccer

T-Pose Soccer is on alpha stage currently only for Windows and is available for free to download. New features are added constantly. So go ahead and download. What are you waiting for?

Changelog

Changes are made and new features are added to T-Pose Soccer often. You can read about the version history here.

    0.5 (Summer World Cup 2018 edition)

  • Added speed bars to UI for both players which are limiting the player's run power.
  • Replaced main menu and gameplay music.
  • Added new sounds.
  • Players blink now after health losing.
  • Added stats system and integrated it to UI.
  • Added control support for gamepads.

    0.4

  • Added health system for both players.
  • SupaBear character added and integrated it to MysteryBox gameobject.
  • Bulldozer character added and integrated it to MysteryBox gameobject.
  • Scrolling bannerboard text includes now JSON functionality. Reads location and weather info based on network connection and parses the data into scrollable text.

    0.2

  • Added intro counter to match start.
  • Tweaked MysteryBox gameobject with new features

    0.1

  • Initial release. T-Pose Soccer is born.

Tools

T-Pose Soccer is made with Unity game development platform, C# with Visual Studio 2017 and MagicaVoxel voxel art editor.
